¿Qué diferencias y semejanzas hay entre Java y JavaScript?

Diferencias: Java es un lenguaje de programación OOP, mientras que Java Script es un lenguaje de scripts OOP. Java crea aplicaciones que se ejecutan en una máquina o explorador virtual, mientras que el código JavaScript sólo se ejecuta en un explorador. El código Java necesita compilación, mientras que el código JavaScript está en todo el texto. Necesitan diferentes plugins. Semejanzas: Ambos sirven para desarrollar soluciones a problemas Ofrecen la opcion para conectarse a bases de datos aunque de diferentes formas Se pueden crear interfaces graficas con ellos
